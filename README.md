# La_Bonne_Fortune
 Tableau de bord du bonheur à long terme

 # Objectif:
 * Prendre de meilleures décisions pour être heureux à long terme
 * Visualiser ce qui est important à long terme
 * Avoir le plaisir de voir les choses grandir
